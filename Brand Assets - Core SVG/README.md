# FieldWise Core SVG Assets

- `fieldwise-core-stacked.svg` — dashboard hero, sign-in and vertically constrained layouts.
- `fieldwise-core-primary.svg` — horizontal navigation headers, website references and wide layouts.
- `fieldwise-core-icon.svg` — compact toolbar and in-app brand mark.

The Core Xcode asset catalogue includes these as:

- `FieldWiseLogo`
- `FieldWisePrimaryLogo`
- `FieldWiseIcon`

All assets preserve their vector representation and render as original artwork rather than template-tinted symbols.
